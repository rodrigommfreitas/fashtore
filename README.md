# [Fashtore](https://rodrigommfreitas.github.io/fashtore/)
**Responsive** clothing store including a *shopping cart* functionality.

## Technologies and tools used:
- [React](https://reactjs.org/), [React Router](https://reactrouter.com/en/main) and [React Query](https://react-query-v3.tanstack.com/);
- [TypeScript](https://www.typescriptlang.org/);
- [Tailwind CSS](https://tailwindcss.com/);
- [Axios](https://axios-http.com/);
- [Vite](https://vitejs.dev/);
- [ESLint](https://eslint.org/);
- [Prettier](https://prettier.io/).

## API
This project uses the RESTful [Fake Store API](https://fakestoreapi.com/) by [Mohammad Reza Keikavousi](https://github.com/keikaavousi).

### Setup
```
npm install
```

### Run
```
npm run dev
```

### Build
```
npm run build
```
